Algerian Forest Fires Prediction

SIT Pune — Under Dr. Satish Kumar V.C. Academic Project Team of 4

Project Overview

This project applies Machine Learning classification techniques to predict whether a forest fire will occur in the Algerian region based on meteorological and environmental features. The dataset contains observations from two Algerian regions (Bejaia and Sidi Bel-Abbes) with weather-based Fire Weather Index (FWI) system components.

We implemented and compared multiple classification models, ultimately building a Stacking Ensemble Classifier that combines the strengths of Logistic Regression, Support Vector Machine (SVM), and Random Forest to achieve state-of-the-art prediction performance.

Extensive EDA was performed to understand feature distributions, correlations, and class separability. The correlation heatmap reveals strong positive correlations between temperature, FFMC, ISI, and FWI — which are key predictors of fire occurrence. Humidity (RH) shows a strong negative correlation with fire risk.

Model Architecture

Base Models

Three base classifiers were individually trained and evaluated:

  • Logistic Regression — A linear classifier using the sigmoid function. Effective as a baseline and interpretable model.
  • Support Vector Machine (SVM) — Uses a radial basis function (RBF) kernel to find an optimal hyperplane in high-dimensional space for non-linear classification.
  • Random Forest — An ensemble of decision trees using bagging. Provides strong individual performance and feature importance rankings.

Stacking Ensemble (Final Model)

The stacking ensemble uses all three base models as Level-0 learners. Their predictions are fed as features to a Logistic Regression meta-learner (Level-1), which learns the optimal combination of base model predictions. This architecture captures both linear and non-linear patterns in the data.

Cross-validation was used during stacking to prevent data leakage — base model predictions for the meta-learner are generated out-of-fold.
